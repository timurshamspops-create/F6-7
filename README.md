# F6-7 League — сайт чемпионата

Сайт симрейсинг-лиги: результаты гонок, таблица лидеров, гонщики, команды, ресурсы и админ-панель.
Состоит из двух частей:

- **client/** — сайт (React), собирается в статичные файлы
- **server/** — маленький сервер на Node.js, который отдаёт сайт и хранит данные в файле `server/data.json`

Пароль администратора хранится в `server/config.json` (создаётся автоматически при первом запуске, по умолчанию `F67-2026`) и проверяется на сервере, а не в браузере.

---

## 1. Что нужно установить один раз

**Node.js** (LTS-версия): https://nodejs.org — скачайте установщик, поставьте с настройками по умолчанию.

Проверить, что всё установилось — откройте PowerShell или cmd и введите:
```
node -v
npm -v
```
Должны показаться номера версий.

---

## 2. Установка и запуск сайта

1. Распакуйте архив с сайтом в любую папку, например `C:\f6-7-league-site`
2. Дважды кликните **install.bat** — он установит всё нужное и соберёт сайт (займёт пару минут, нужен интернет)
3. Дважды кликните **start.bat** — запустится сервер
4. Откройте в браузере **http://localhost:3000** — сайт должен открыться

Пока открыто окно `start.bat`, сайт работает. Закрыли окно — сайт выключился.

### Сменить пароль администратора
Откройте `server/config.json` в блокноте, замените значение `adminPassword`, сохраните файл, перезапустите `start.bat`.

### Данные сайта
Все гонщики, команды, гонки и ссылки хранятся в `server/data.json`. Полезно иногда копировать этот файл куда-то ещё — это ваша единственная резервная копия.

## 3. Подключение домена (у вас уже есть домен)

Ваш ПК должен быть доступен из интернета. Домашний IP обычно динамический и меняется, плюс открывать порты на роутере — не самый безопасный вариант. Поэтому рекомендую **Cloudflare Tunnel**: он даёт домену HTTPS и связь с вашим ПК без проброса портов.

### Шаг 1 — перенести домен на Cloudflare (бесплатно)
1. Зарегистрируйтесь на https://dash.cloudflare.com
2. Нажмите **Add a site**, введите ваш домен
3. Выберите бесплатный тариф
4. Cloudflare покажет два адреса нейм-серверов (например `ada.ns.cloudflare.com`)
5. Зайдите в панель управления у регистратора, где куплен домен, и замените нейм-серверы на выданные Cloudflare
6. Подождите — обычно применяется от 10 минут до нескольких часов

### Шаг 2 — установить cloudflared на Windows
Через winget (Windows 10/11):
```
winget install --id Cloudflare.cloudflared
```
Или скачайте установщик вручную: https://github.com/cloudflare/cloudflared/releases (файл `cloudflared-windows-amd64.msi`)

### Шаг 3 — авторизоваться и создать туннель
В PowerShell:
```
cloudflared tunnel login
```
Откроется браузер — выберите там свой домен.

```
cloudflared tunnel create f67-league
```
Запомните UUID туннеля, который выведется в консоли.

### Шаг 4 — настроить туннель
Создайте файл `%USERPROFILE%\.cloudflared\config.yml`:
```yaml
tunnel: f67-league
credentials-file: C:\Users\ВАШ_ПОЛЬЗОВАТЕЛЬ\.cloudflared\<UUID туннеля>.json

ingress:
  - hostname: league.вашдомен.ru
    service: http://localhost:3000
  - service: http_status:404
```
Замените `league.вашдомен.ru` на то, что хотите (можно и сам корневой домен `вашдомен.ru`).

### Шаг 5 — привязать домен к туннелю
```
cloudflared tunnel route dns f67-league league.вашдомен.ru
```

### Шаг 6 — запустить туннель
```
cloudflared tunnel run f67-league
```
Держите это окно открытым вместе с `start.bat` — либо установите туннель как фоновую службу Windows, чтобы он запускался сам:
```
cloudflared service install
```

Теперь сайт открыт по адресу `https://league.вашдомен.ru`, пока работают `start.bat` и туннель, а ваш ПК включён и подключён к интернету.

### Альтернатива: проброс портов на роутере
Более старый способ — в настройках роутера пробросить порт 80 (и 443 для HTTPS) на локальный IP вашего ПК, и настроить Dynamic DNS (например DuckDNS или No-IP), так как домашний IP обычно меняется. Это менее удобно и более уязвимо для атак, поэтому Cloudflare Tunnel обычно предпочтительнее.

## Частые проблемы

- **Сайт открывается, но пустой / ошибка «Не удалось связаться с сервером»** — сервер (`start.bat`) не запущен или упал. Проверьте окно консоли на ошибки.
- **install.bat падает с ошибкой** — скорее всего не установлен Node.js или нет интернета.
- **Сайт недоступен снаружи по домену** — проверьте, что нейм-серверы домена реально переключены на Cloudflare (можно проверить на whatsmydns.net), и что `cloudflared tunnel run` запущен.
